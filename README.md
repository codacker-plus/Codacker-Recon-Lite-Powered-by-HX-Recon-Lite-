# Codacker Recon Lite

Codacker Recon Lite is a simplified reconnaissance tool designed for learning and practical use in penetration testing workflows.

This project is adapted for the Codacker community and focuses on usability, clarity, and real-world application.

---

## Overview

Reconnaissance is one of the most important phases in penetration testing, but it often involves running multiple tools manually and managing scattered outputs.

Codacker Recon Lite solves this by organizing the workflow into a structured and repeatable process.

---

## Features

- Multi-source subdomain enumeration  
- Alive host detection  
- Clean and organized output files  
- Project-based workflow  
- Beginner-friendly CLI interface  

---

## Workflow

Target  
→ Subdomain Discovery  
→ Alive Check  
→ Output Storage
